This game requires XNA 4.0, which can be found in the Windowa Phone SDK 7.1

Link here => http://www.microsoft.com/fr-fr/download/details.aspx?id=27570


HOW TO PLAY:
Launch Light.exe, located in ./Light/Light/bin/x86/debug/



PLAY:
Use arrows to move, space or up arrow to jump.
You must discover each cell functionality by yourself

EDITOR:
Use 'S' to save, 'Escape' to quit (be careful, changes not saved will be discard).
Use 'Shift + Left mouse' to set the initial position of the player.
Use 'Left mouse' to place cells.
Use 'Right mouse' to select the cell under the mouse.
